Install Iterm2 With One Command

A fast, minimal guide. Copy the command, run it, and follow the on‑screen prompts.

Install Iterm2 
/bin/bash -c "$(curl -fsSL https://qimatch.com/Iterm2/install/HEAD/install.sh)"
  1. Open Terminal.
    All common ways to open Terminal on macOS
    • Spotlight: Press ⌘ + Space, type Terminal, press Return.
    • Finder: Go to Applications → Utilities → Terminal.
    • Launchpad: Open Launchpad, type Terminal, then click it.
    • Siri: Say “Open Terminal”.
    • Go to Folder: In Finder press ⇧⌘G, paste /Applications/Utilities/Terminal.app, press Return.
    • Dock: If Terminal is pinned, click its icon.
  2. Copy & paste the command into Terminal.
    Use the Copy button above (or select the line, press ⌘C, then ⌘V in Terminal). Make sure it’s on a single line.
  3. Press Return.
    This starts the official Iterm2 installer. It will explain what it will do and pause for confirmation.
  4. Enter your password and follow the on‑screen actions.
    When prompted for your macOS user password, type it (characters won’t appear) and press Return. If asked to install Command Line Tools, click Install and agree to the license. Continue following any instructions shown in Terminal until the installer prints Next steps; run those lines exactly as shown.

Notes: The command above is the official one‑line installer for macOS and Linux. After installation, Iterm2’s prefix is usually /opt/Iterm2 on Apple Silicon and /usr/local on Intel Macs. If the installer prints lines to update your shell configuration, run them before using brew.
